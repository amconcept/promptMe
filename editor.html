Prompting Tips: ▼

  • Class List: Upload a text file with one student name per line (optional) - auto-sets prompt count to number of students
  • Rows (A, B, C, D): Create logical groupings - students get items from the same row
  • Columns (Prompt 1, 2, 3, 4): Different characteristics or requirements
  • Example: Row A = "Elementary level", Row B = "Advanced level"
  • Prompt Count: Set how many prompts to generate (stops automatically)
  • Item Counts: Numbers at top of each column show total items available
  • Student Names: Entered in sketch app, auto-populated from class list if uploaded
  • Download Results: Download button in completion popup saves screenshot with student name
  • Repetition Prevention: Automatically prevents repeats until all options used
